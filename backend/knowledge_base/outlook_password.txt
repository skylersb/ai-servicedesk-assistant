Title:
Fix Outlook sign-in issues

Problem:
User cannot sign into Outlook or is repeatedly prompted for credentials.

Applies To:
Outlook desktop, Outlook mobile, Microsoft 365

Steps:
1. Confirm your password works at:
   https://portal.office.com

2. Close Outlook completely
3. Reopen Outlook and sign in again

4. If issue persists:
   - Go to Control Panel → Credential Manager
   - Remove saved Outlook/Office credentials
   - Restart Outlook

Troubleshooting:
- Password keeps failing → Reset password
- Account locked → Wait and retry
- Mobile issue → Remove and re-add account

Escalation:
If login issues continue after reset, contact IT for account review.
